# MyBakingRecipe
[My Baking Recipe](https://github.com/DavidYang2149/MyBakingRecipe)는 레시피를 작성하고 저장하는 공간입니다.

## Install dependencies

```sh
npm install
```

## Run dev server

```sh
npm run dev
```

## Run deploy in github

```sh
npm run deploy
```

## Lint and fix

```sh
npm run lint
```

## Run e2e tests

```sh
npm run test:e2e
```

## Run Coverage

```sh
npm run coverage
```
